ffmpeg

Posted on 2021-07-30 Edited on 2021-08-01

日常用到的命令汇总

查看视频基本信息
ffprobe input.mp4

提取音频
ffmpeg -i input.mp4 -vn -c copy output.m4a

将直播源 url 转录到 output.ts
ffmpeg -i "url" -c copy "output.ts"

连接多个音频
ffmpeg -f concat -i filelist.txt -c:a copy output.mp3
filelist.txt 中包含要连接的音频，例如：
