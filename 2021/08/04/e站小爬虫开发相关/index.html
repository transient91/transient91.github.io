e站小爬虫开发相关

Posted on 2021-08-04 Edited on 2021-08-06

总结一下用到的知识点。

用到的库：

retrying

这个包提供了一个装饰器函数 retry，可以在出错时重试被装饰的函数。
更多用法python 的 retrying 库处理尝试多次请求

imghdr

通过文件或字节流判断图片格式。
具体见python 模块之 imghdr 检测图片类型

tqdm

一个进度条库，只用到了最基础的功能，后续可以美化。

fake_useragent

用于伪造 user-agent 反反爬。

json

python 自带的 json 解析库。

改成

可以避免这个问题，不过博客嘛记录一下这个问题好了。

下面这个函数可以修改 config.json 中的 key 的 value。

另外还有对应的 loads 和 dumps 方法，用于变量间解析数据。
总的来说：
json.load()从文件中读取 json 字符串
json.dump()将 json 格式字符串写到文件中
json.loads()将 json 字符串转换为字典类型
json.dumps()将 python 中的字典类型转换为字符串类型
具体参考：python 中 json.load()、json.loads()、json.dump()、json.dumps()的区别

getopt

有:的参数如果出现一定要有值，不然报错。
突然想到可以 except 这个异常来实现如果没有值就用默认值。下次试试。

re

用 group 方法获取匹配到的字符串，其中 group(0)是整个匹配到的串（例如这里可能是"Showing 1 - 10 of 20 images"），之后就是括号里的串了（按刚刚的例子 group(1)就是"20"）。

BeautifulSoup

find_all 方法返回匹配到的所有 tag 组成的列表，用正则表达式可以匹配含有这个属性的 tag（例如这里匹配形如的 tag，其中 value 含有 url 即可）。
